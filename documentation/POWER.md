# Documentation Power

## Overview

Direct access to comprehensive AWS documentation, including service guides, API references, best practices, and tutorials.

## Available MCP Servers

### aws-documentation
**Package:** `awslabs.aws-documentation-mcp-server@latest`
**Purpose:** Search and retrieve AWS documentation
**Configuration:**
- `AWS_DOCUMENTATION_PARTITION` - AWS partition (aws, aws-cn, aws-us-gov)

## Best Practices

### ✅ Do:
- Search documentation before implementation
- Reference official guides for best practices
- Check API documentation for parameters
- Review service limits and quotas
- Read security best practices

### ❌ Don't:
- Don't rely on outdated documentation
- Don't skip reading service-specific guides
- Don't ignore security recommendations

## Tips

1. **Search effectively** - Use specific service names
2. **Check examples** - Look for code samples
3. **Review limits** - Understand service quotas
4. **Security first** - Read security best practices
5. **Stay updated** - Check for service updates
